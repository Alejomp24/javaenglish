//El WHILE es muy parecido al IF en su sintaxis y la diferencia es que el WHILE se ejecuta varias veces mientras se cumpla la condición.

int i = 0
while (i<=24) {
	System.out.println(i);
	i++;
}

//Si no se especifica el incremento el ciclo queda pegado. (en loop infinito)

//DO WHILE

do {
	System.out.println(i + " ");
	i++;
} while (i<5)

//La diferencia entre WHILE y DO WHILE es la condición, en el WHILE la condición está de primera y en el DO WHILE la condición está de última.
//Por ejemplo:
jshell> int i = 10
i ==> 10

jshell> while (i < 5) {
   ...>     System.out.print(i + " ");
   ...>     i++;
   ...> }
