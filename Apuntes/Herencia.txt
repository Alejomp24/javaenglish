CLASS ABSTRACT
Tiene una característica mediante la cual no es necesario específicar implementaciones de algunos metodos, es decir,
metodos que no tienen definición, no tienen cuerpo. No dicen lo que hacen, solo tienen la declaración.

jshell> abstract class AbstractAnimal{
   ...>     abstract public void bark();
   ...> }
|  created class AbstractAnimal


Para una clase abstracta no se pueden crear instancias, sin embargo, se pueden extender clases abstractas y crear clases concretas.

jshell> class Dog extends AbstractAnimal{
   ...>     public void bark(){
   ...>         System.out.println("Gua Gua");
   ...>     }
   ...> }
|  created class Dog

jshell> Dog dog = new Dog();
dog ==> Dog@29ca901e

jshell> dog.bark()
Gua Gua
