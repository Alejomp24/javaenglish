CREAR CLASS
-En el Proyecto, en la carpeta src, dar click derecho y New -> Class
-Se indica el nombre de la Clase y se sugiere un package que en este caso se utilizó com.in28minutes.firstjavaproject
-En los metodos para este caso se seleccionó public static void main(String[] args) e Inherited abstract methods

REALIZAR UN DEBUG
-Se debe colocar un Break Point: 
 Para ello se debe dar doble click en la franja azul asociado a la línea de código que se quiere revisar.

-Step Over:
 Es ejecutar la linea de código actual y pasar a la siguiente linea de código, pasando por encima alguna invocación de código
 Por ejemplo: 
	*6	MultiplicationTable table = new MultiplicationTable();     // Aquí está el Break Point, se completa la linea 6, no entraría hacer el paso a paso del proyecto MultiplicationTable y pasaría a la linea 7   
	 7	table.print(2, 3, 8);

-Step Into:	 
 Es ejecutar todas las lineas de código del software
 Por ejemplo, para el caso anterior en la linea 6 se ejecutarian las lineas de código del proyecto MultiplicationTable 
 
-Run to Line:
 Es ejecutar el programa hasta la linea de código que le señalemos. 

-Generar Código Automáticamente.
 Click derecho -> Source -> Generate Getters & Setters
