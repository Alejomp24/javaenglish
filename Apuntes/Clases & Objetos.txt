CLASES & OBJETOS
Ejemplo. CLASE: País - OBJETOS: Colombia, Inglaterra, USA

CREAR UNA CLASE (TEMPLATE)
jshell> class Country{
   ...> }
|  created class Country

CREAR UN OBJETOS (INSTANCIA)
jshell> Country colombia = new Country()
colombia ==> Country@5702b3b1

CREAR METODO EN UNA CLASE
jshell> class Planet{
   ...>     void revolve(){
   ...>         System.out.println("Revolve");
   ...>     }
   ...> }
   ...>
|  created class Planet

jshell> Planet earth = new Planet()
earth ==> Planet@73846619

jshell> Planet venus = new Planet()
venus ==> Planet@29ca901e

EJECUTAR METODO DE UNA CLASE
Sintaxis
Objeto.Metodo()
jshell> earth.revolve()
Revolve

**Es necesario que la Clase tenga datos (en este caso la clase debería tener objetos), de lo contrario sucede lo siguiente:
jshell> marte.revolve()
|  Error:
|  cannot find symbol
|    symbol:   variable marte
|  marte.revolve()
|  ^---^
