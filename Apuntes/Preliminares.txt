DEFINICIONES.

JSHELL: Herramienta para pruebas y evaluación de código interactivo 
JVM: Java Virtual Machine, el entorno de tiempo de ejecución donde se ejecuta el código de bytes de Java (bytecode).
JRE: Java Runtime Environment, necesario para ejecutar programas Java. JRE = JVM + Librerias + Otros Componentes
JDK: Java Development Kit, necesario para compilar y correr programas de Java. JDK = JRE + Compiladores + Depuradores
IDE: Integrated Development Environment, una aplicación de software que ayuda a desarrollar código Java de manera eficiente
