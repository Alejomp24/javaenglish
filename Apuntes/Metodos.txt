DEFINICIONES
	Metodo: Un bloque de código con nombre que realiza una tarea específica y se puede llamar varias veces.
	Clase: un plan para crear objetos en Java
	Objeto: una instancia de una clase Java que tiene una identidad, comportamiento y estado 

SINTAXIS METODO
ReturnType nameOfTheMethod(Type argumentName){
  //Body of the method, What do we want to do in the method?
}
*El argumentName vendría siendo los parámetros de entrada del metodo, entradas proporcionadas a un método durante la llamada al método
*El ReturnType vendría siendo al parámetro de salida del metodo, el tipo de datos del valor devuelto por un método Java
	- Void: un tipo de devolución que indica que un método no devuelve un valor

EJERCICIO TABLA DE MULTIPLICAR
void printMultiplicationTable(){
    for(i=1; i<=10; i++){
        System.out.printf("%d * %d = %d", 6, i, 6*i).println();
    }
}

void printMultiplicationTable(int number){
     for(i=1; i<=10; i++){
         System.out.printf("%d * %d = %d", number, i, number*i).println();
     }
}
